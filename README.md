# BetikFetch
BetikSonu's Small-Speedy Shell Fetch


Oto Setup :

    python3 setup.py
 
    [ReOpen your shell]

Manuel Setup 

    chmod +x betikfetch.py
    cp betikfetch.py /usr/bin/betikfetch
    echo -e "\nbetikfetch\n" >> $HOME/.bashrc
